# 고독한 미식가 
고독한 미식가는 위치, 텍스트, 이미지 기반의 다이어리 앱입니다. ([Google Play Store](https://play.google.com/store/apps/details?id=org.ybk.fooddiaryapp))<br>
인스타그램에 음식 사진만 올리는 사람들을 보고 개발하게 되었습니다.<br>
<br>

## Skills
`Android Jetpack`(`ViewModel`, `LiveData`, `DataBinding`, `Navigation`), `Firebase`(`Authentication`, `Realtime Database`, `Storage`), `Retrofit`, `RxJava`, `Dagger`, `Mockito`, `hamcrest`, `Espresso`, `JUnit`<br>
