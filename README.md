# 고독한 미식가 [Google Play Store](https://play.google.com/store/apps/details?id=org.ybk.fooddiaryapp)

## Introduce.

> 고독한 미식가는 그날에 갔던 맛집의 위치와 그 곳에서 먹은 음식을 다이어리로 기록할 수 있습니다.

### :point_right: Why
> 인스타그램에 음식 사진만 올리는 사람들을 보고 영감을 받아 개발했습니다.
